# CV-BIO-DATA

Static GitHub Pages site for:

`https://bbdmahbub.github.io/CV-BIO-DATA/`

## Publish Steps

This folder is not a Git repository yet. Run these commands from this project folder:

```powershell
git init
git branch -M main
git add .
git commit -m "Set up GitHub Pages deployment"
git remote add origin https://github.com/bbdmahbub/CV-BIO-DATA.git
git push -u origin main
```

## GitHub Settings

1. Create the GitHub repository named `CV-BIO-DATA` under the `bbdmahbub` account.
2. Push this folder to that repository.
3. Open `Settings -> Pages`.
4. Set the source to `GitHub Actions`.
5. Wait for the `Deploy GitHub Pages` workflow to finish.

After that, the site URL will be:

`https://bbdmahbub.github.io/CV-BIO-DATA/`
